The Direction

Painted tabletop adventure

Gouache color, inked silhouettes, tactile materials, and enough exaggeration to remain clear when a tile is only 32 pixels wide.

Design Rules

What stays consistent

01

Readable first

One dominant silhouette per tile, tested at actual game size.

02

Tactile pieces

Ships, pirates, and coins should feel like objects on a table.

03

Tropical color

Turquoise, teak, parchment, coral, jungle green, and brass.

04

No visual noise

Texture supports the icon; it never competes with game state.

Feedback

Review by asset name

Send feedback in our chat using the asset name and what should change. For example: Crocodile: friendlier face, less dark water.
